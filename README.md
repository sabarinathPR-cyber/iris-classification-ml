# Iris Flower Classification

This project uses Machine Learning to classify Iris flowers into three species:
Setosa, Versicolor, and Virginica.

The model is trained using a large dataset and the K-Nearest Neighbors (KNN) classification algorithm.

## Project Overview

The Iris dataset contains measurements of flower features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Using these features, the model predicts the species of the iris flower.

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Machine Learning

## Machine Learning Algorithm

K-Nearest Neighbors (KNN) Classification

## Dataset

Large Iris dataset with **5000 records** used for training and testing the model.

Dataset file:iris_dataset_large_5000.csv
